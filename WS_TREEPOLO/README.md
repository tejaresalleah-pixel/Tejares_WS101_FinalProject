# Title: Design of Treepolo Café’s Cozy Hangout Promotional Website

**Designer:** Alleah Wendine M. Tejares – 3A

## Project Overview
    This project focuses on designing a promotional website for **Treepolo Café**, a nature-inspired dining destination in Antique known for its rustic ambiance and relaxing “al fresco” experience. The website is targeted at local coffee enthusiasts, digital nomads, and travelers seeking a peaceful, aesthetically pleasing hangout.

    The layout emphasizes a cozy and organic visual style. It showcases Treepolo’s signature beverages, comfort food, and lush environment through structured sections, including the Menu, Events/Services, Gallery, and Contact details. The primary goal is to create an inviting digital presence that accurately reflects the café’s physical atmosphere and encourages online visitors to experience it in person.

## Key Features
**Organic & Cozy UI/UX:** A modern, nature-inspired design utilizing warm color palettes and elegant typography.<br>
**Scroll Animations:** Smooth reveal animations built natively using JavaScript (`IntersectionObserver`) and CSS transitions for a lightweight, performant experience.<br>
**Interactive Elements:** Dynamic popups, responsive navigation menus, and clean layout structures for cross-device compatibility.<br>
**Structured Content Sections:** Easy-to-navigate Menu, Gallery, and Contact hubs tailored for customer convenience.<br>

## Technologies & Frameworks Used:
**HTML5** – Semantic structure and content.<br>
**CSS3** – Custom styling, responsive layouts, and native transition effects.<br>
**JavaScript (Vanilla)** – Interactive UI components, modal popups, and native scroll-reveal functionality (`IntersectionObserver`).<br>
**Font Awesome** – Scalable vector icons for UI elements.<br>
**Google Fonts** – Playfair Display and Poppins for elegant, highly readable typography.<br>

## Design Tools & AI Integration

    To streamline development and refine the design, Artificial Intelligence and standard design tools were utilized throughout the project lifecycle.

### Tools Used
**1. ChatGPT:** Utilized for debugging HTML/CSS and improving JavaScript functionality, specifically focusing on UI fixes, responsive layout structures, and interactive features (ex., popups and scroll animations).<br>
**2. Gemini:** Used to generate textual content, define the color palette, and guide the overall user experience and design flow.<br>
**3. Canva:** Used for processing visual assets, image curation, and supplementary design elements.<br>
**4. Pinterest:** Used as a reference board for layout inspiration and identifying the specific "rustic café" aesthetic direction.
